Visual Studio Code C/C++配置指南

一.准备工作

1.安装MinGW编译器

在官网上下载MinGW编译器，并进行环境变量的配置。

2.安装Visual Studio Code的C/C++插件

直接在Visual Studio Code搜索C/C++ Extension Pack(C/C++扩展包)安装即可

二.Visual Studio Code配置工作

切记在配置运行环境时，请始终保持Visual Studio Code将C/C++源文件在编辑器里处于编辑状态，这样Visual Studio Code会根据打开的源文件类型智能推荐配置工作和配置文件中编译程序路径的自动补全。

1.创建C/C++源文件

创建任意C/C++源文件，并键入HelloWorld代码

2.配置生成任务

点击终端->配置默认生成任务，然后根据编译器提供的选择生成程序，在MinGW中C使用GCC,C++使用G++

点击终端，运行生成任务(Ctrl+F9)，将C/C++文件编译为可执行文件，若有错误根据报错进行修改。必须先执行生成任务后，才可进行调试！

3.配置调试文件

保持C/C++源文件处于编辑状态，点击运行->添加配置，选择C/C++(GDB/LLDB)，然后选择调试程序(C选择GCC,C++选择G++),然后Visual Studio Code会自动开启调试。
